# japanese-tutor
음성으로 대화하면서 일본어를 가르쳐주는 선생님입니다. Whisper API와 OpenAI API를 사용했습니다.

## 사용법

1. OpenAI API Key를 발급받습니다.
2. OPENAI_API_KEY 환경변수를 등록합니다.
3. 패키지를 설치합니다.

    ````
    $ pip install -r requirements.txt
    ````

4. 실행합니다.

    ````
    $ python3 main.py
    ````
